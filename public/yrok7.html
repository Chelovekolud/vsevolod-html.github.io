Вот мы и продолжаем серию уроков, надеюсь вы прошли зачёт. В этом уроке мы ещё немного посмотрим на html и перейдём к css.

Кроме того что мы пишем теги, мы можем писать и классы, например:

  1. <p class="class1">
  2. я тег с классом
  3. </p>

В коде выше мы присвоили тегу параграф класс class1, заметьте что название класса должно быть на латинице и не начинаться с цифры, а главное что нет никаких символов. Иначе класс будет не работать!

В css класс мы пишем начиная с точки, ведь без точки можно только теги.

  1. .class1 {
  2. color: red;
  3. }

Выше мы присвоили всем тегам с определённым классом красный цвет. Бывает что надо присвоить не всем одинаковым тегам, а только определённым например красный цвет, в этом случае нам нужны классы.

Кроме классов есть id для css настроек его используют реже, но всё же покажу пример:

HTML

  1. <p id="paragraf0id>
  2. Параграф с id
  3. </p>

CSS

  1. #paragraf1id {
  2. color: green;
  3. }

Но id чаще используют для ссылок в одной странице, думаю все видели как по нажатию на ссылку, страница промывается в конец или начало. Это так называемый якорь, вот вам пример:

<a href="#paragraf1id ">Якорь ⚓</a>
